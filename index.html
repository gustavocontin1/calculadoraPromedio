Calculadora de Promedios de Notas

¿Como sacar un promedio?

La Calculadora de Promedio se usa para calcular el valor promedio de cualquier conjunto de números.

Cálculo del valor promedio

Supongamos que tenemos un espacio de muestra a1, a2, ...., an. A continuación, el valor medio se define mediante la siguiente ecuación:

Promedio = (a 1 + a 2 + ... + a n ) / n
